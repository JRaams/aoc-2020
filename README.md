## 1. Running the code

#### 1.1. Requirements

1. Golang (tested with v1.15.5) [Download and install](https://golang.org/doc/install)

#### 1.2. Steps

1. Change dir to the desired day (Assuming you are in folder 'aoc-2020'): `$ cd calendar/day7`

2. Run main.go:
   `$ go run .`
