## 1. Running the code

#### 1.1. Requirements

1. Golang (tested with v1.15.5) [Download and install](https://golang.org/doc/install)

#### 1.2. Steps

1. Change dir to the desired day, e.g.
   (Assuming you are in 'aoc-2020'): `$ cd src/day1`

2. Run main.go:
   `$ go run main.go`
